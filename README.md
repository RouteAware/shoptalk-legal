# ShopTalk Legal Documents

GitHub Pages site hosting Privacy Policy and Terms of Service for the ShopTalk mobile app.

## What's Included

- **Privacy Policy** - Comprehensive GDPR/CCPA compliant privacy policy
- **Terms of Service** - Community guidelines and terms of service
- **Landing Page** - Simple navigation to both documents

## URLs (After GitHub Pages Setup)

Once deployed to GitHub Pages, these documents will be accessible at:

- Landing Page: `https://[your-username].github.io/shoptalk-legal/`
- Privacy Policy: `https://[your-username].github.io/shoptalk-legal/privacy-policy.html`
- Terms of Service: `https://[your-username].github.io/shoptalk-legal/terms-of-service.html`
